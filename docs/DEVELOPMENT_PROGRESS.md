## 🎮 **快速开始**

### **1. 启动后端服务器**
```bash
cd ai-collaboration-v2/backend

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 或使用启动脚本
chmod +x start.sh
./start.sh dev
```

### **2. 运行演示**
```bash
# 在新的终端运行演示
cd ai-collaboration-v2/scripts
node demo.js
```

### **3. 测试API**
```bash
# 健康检查
curl http://localhost:4000/health

# 系统信息
curl http://localhost:4000/api/system

# 获取AI代理列表
curl http://localhost:4000/api/agents
```
